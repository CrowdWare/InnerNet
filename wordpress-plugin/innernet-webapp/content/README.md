Hier kannst du zusätzliche Inhalte für die Webapp ablegen (z. B. SML-Dateien, Bilder, weitere Assets).

Alle Dateien in diesem Verzeichnis sind über `window.INNERNET_CONTENT_BASE` erreichbar, z. B.:

```
${window.INNERNET_CONTENT_BASE}dein-file.sml
${window.INNERNET_CONTENT_BASE}assets/logo.png
```
